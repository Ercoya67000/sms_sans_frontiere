# SMS Sans Frontière

Passerelle SMS Android permettant à un logiciel sur PC d'envoyer des SMS via un téléphone Android connecté au réseau local.

## Objectifs

- Envoi de SMS via une carte SIM
- Communication sécurisée avec un logiciel Python
- Fonctionnement sur le réseau local (Wi-Fi)
- Historique des SMS
- Application légère et open source
